The Formula

Score = Cosine_Sim × Ebbinghaus_Strength

Memory strength is calculated using time elapsed, initial importance, and recall frequency. Every time you mention a topic, it gets reinforced. If you don't, it quietly fades away.

1. Docker Installation

git clone https://github.com/sachitrafa/cognitive-ai-memory.git && cd cognitive-ai-memory && docker compose up -d

2. Add to Claude (MCP)

{

      "command": "yourmemory"
    }
  }
}
