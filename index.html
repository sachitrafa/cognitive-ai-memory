Three commands.

Zero infrastructure. DuckDB database created automatically at ~/.yourmemory/memories.duckdb on first use.

1. Install
$ pip install yourmemory
2. Get your Cline / Cursor config
$ yourmemory-path
Prints your full executable path + ready-to-paste JSON config for Cline, Cursor, Windsurf, and more. VS Code doesn't inherit shell PATH — this fixes it.
3. Claude Code — add to ~/.claude/settings.json
{
  "mcpServers": {
    "yourmemory": {
      "command": "yourmemory"
    }
  }
}

PostgreSQL (optional — teams / large datasets)

Install with Postgres support
$ pip install yourmemory[postgres]
Create a .env file
DATABASE_URL=postgresql://YOUR_USER@localhost:5432/yourmemory

Backend selected automatically — postgresql:// → Postgres + pgvector, anything else → DuckDB.
